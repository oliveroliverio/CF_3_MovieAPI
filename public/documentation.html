Get All Movies

Description Returns a list of all movies.
URL /movies
Method GET
Query Params None
Request Body None
Response JSON array of movie objects

Example Request

GET /movies HTTP/1.1
Host: localhost:8080

Example Response

[
  {
    "title": "Blade Runner",
    "director": "Ridley Scott"
  },
  {
    "title": "The Matrix",
    "director": "Lana Wachowski, Lilly Wachowski"
  }
]

Get Movie by Title

Description Returns details of a specific movie by title.
URL /movies/:title
Method GET
Query Params title (string)
Request Body None
Response JSON object of the movie

Example Request

GET /movies/Blade%20Runner HTTP/1.1
Host: localhost:8080

Example Response

{
  "title": "Blade Runner",
  "director": "Ridley Scott"
}

Add a New Movie

Description Adds a new movie to the list.
URL /movies
Method POST
Query Params None
Request Body JSON object of the new movie
Response JSON object of the added movie

Example Request

POST /movies HTTP/1.1
Host: localhost:8080
Content-Type: application/json

{
  "title": "Inception",
  "director": "Christopher Nolan"
}

Example Response

{
  "title": "Inception",
  "director": "Christopher Nolan"
}

Update Movie by Title

Description Updates details of a specific movie by title.
URL /movies/:title
Method PUT
Query Params title (string)
Request Body JSON object of the updated movie
Response JSON object of the updated movie

Example Request

PUT /movies/Blade%20Runner HTTP/1.1
Host: localhost:8080
Content-Type: application/json

{
  "title": "Blade Runner",
  "director": "Ridley Scott"
}

Example Response

{
  "title": "Blade Runner",
  "director": "Ridley Scott"
}

Delete Movie by Title

Description Deletes a specific movie by title.
URL /movies/:title
Method DELETE
Query Params title (string)
Request Body None
Response JSON object of the deleted movie

Example Request

DELETE /movies/Blade%20Runner HTTP/1.1
Host: localhost:8080

Example Response

{
  "title": "Blade Runner",
  "director": "Ridley Scott"
}
